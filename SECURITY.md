# Security Policy

## Supported Versions

Security update available at version 1.0.0, after the stdoc stable.
Security provide safe function, every time ISO updates C, we're always updates too, BUT we still compatible with older version of ISO

| Version | Supported          |
| ------- | ------------------ |
| 0.x.x   | not available      |
| 1.x.x   | next update        |
| ...     | not available      |
| ...     | not available      |

## Reporting a Vulnerability

You can report Vulnerability issues in Issues tab or open PR for changing some source codes, stdoc aiming to safety lib, always read the documentations of this lib for deprecation function
